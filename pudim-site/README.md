# Projeto Pudim

Este é um site simples feito em HTML e CSS sobre a sobremesa pudim.

## Descrição

O objetivo do projeto foi criar uma página web básica utilizando HTML5 e CSS3, apresentando informações sobre o pudim, como origem, curiosidades e uma pequena opinião pessoal.

## Tecnologias utilizadas

- HTML5  
- CSS3
